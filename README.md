# concourse-on-docker

## How to use

- Install using the docker composer
```
git clone https://github.com/kiyohome/concourse-on-docker.git
cd concourse-on-docker/latest
./generate-key.sh
docker-compose up -d
```

- Access "http://127.0.0.1:8888/" in the browser
- Login
  - team: main
  - username: concourse
  - password: password
